# Koda — SEO Dashboard

Dashboard administrativo para análise de métricas de SEO e tráfego orgânico. Visualize cliques, impressões, CTR, posicionamento, backlinks e tendências com dados em tempo real.

## Funcionalidades

- Métricas de desempenho com indicadores de tendência (cliques, impressões, CTR, posição)
- Gráficos de performance ao longo do tempo com modo tempo real
- Análise de keywords com filtros de dificuldade e ordenação
- Monitoramento de backlinks e domínios referenciadores
- Análise de tendências históricas
- Interface trilíngue (PT-BR, EN-US, Español) com Dark Mode
- Exportação de dados em CSV
